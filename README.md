stanleyr
========

Stanleyr is a Tumblr port of Black Tieâ€™s Stanley Theme. Like Stanley, Stanleyr is based on Bootstrap 3.0.3 framework. So it is responsive and extendable.

Stanleyr is a really simple theme for those wanting to showcase their work with a cute & clean style.

Features:

- Supports all post type
- Highly responsive
- Free and Open sourced on Github
- Custom colors
- Social Links
- Disqus Comments
- Google Analytics
- Icons fetched from source

Demo- http://stanleyr-theme.tumblr.com/
